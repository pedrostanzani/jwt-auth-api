# Instalação
 
Siga as seguintes instruções para instalar e executar o projeto.


## Instalação local

<Steps>
### Download
 
Faça o download do arquivo `compose.yaml` [neste link](https://raw.githubusercontent.com/pedrostanzani/jwt-auth-api/refs/heads/main/compose.yaml) e armazene-o em um diretório vazio.
 
### Execução com Docker Compose
 
Execute o comando abaixo no terminal do diretório em que o arquivo `compose.yaml` foi armazenado:
```bash
docker compose up
```

### All set!
 
Agora você pode acessar a API em [http://localhost:3000](http://localhost:3000).

</Steps>
